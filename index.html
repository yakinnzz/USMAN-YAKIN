1. PENGERTIAN OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang menggunakan objek sebagai dasar perancangan program. Pendekatan ini memungkinkan kode menjadi lebih terstruktur, modular, dan mudah dikembangkan.

2. Class Dan Object

Class merupakan blueprint atau cetak biru yang digunakan untuk membuat objek. Class berisi atribut (variabel) yang menggambarkan karakteristik suatu objek serta method (fungsi) yang menggambarkan perilaku objek tersebut. Sedangkan object merupakan instansi dari sebuah class.

// Nama   : USMAN YAKIN TRIO SINADA
// Nim    : 251410006
// Kelas  : Si2A
class Car {
    String? model;
    String? color;
    
    void start() { 
        print('Car started!'); 
    }
    void stop() { 
        print('Car stopped!'); 
    }
}

3. CONSTRUCTOR

Constructor adalah method khusus untuk inisialisasi nilai awal pada objek ketika objek dibuat.

// Nama   : USMAN YAKIN TRIO SINADA
// Nim    : 251410006
// Kelas  : Si2A
class Mahasiswa {
  String nama; // nama : USMAN YAKIN TRIO SINADA
  String nim;  // nim  : 251410006
  double ipk;  // ipk  : 3.90

  // Constructor Mahasiswa (Menginisialisasi data input)
  Mahasiswa(this.nama, this.nim, this.ipk);

  // Method untuk menampilkan data mahasiswa ke konsol
  void tampilkanData() {
    print('Nama : $nama');
    print('NIM  : $nim');
    print('IPK  : $ipk');
  }
}

void main() {
  // Membuat objek mhs1 dari class Mahasiswa
  Mahasiswa mhs1 = Mahasiswa("USMAN YAKIN TRIO SINADA", "251410006", 3.90);

  mhs1.tampilkanData(); // Memanggil method tampilkanData
}

5. ENCAPSULATION

Encapsulation membungkus data dan method serta membatasi akses langsung dari luar class.

// Nama   : USMAN YAKIN TRIO SINADA
// Nim    : 251410006
// Kelas  : Si2A
class Person {
  String _name; // Atribut private (diawali underscore)
  int age;      // Atribut publik

  // Constructor
  Person(this._name, this.age);

  // Setter untuk mengubah nilai _name secara aman
  void setName(String name) => _name = name;

  // Getter untuk mengambil nilai _name
  String getName() => _name;
}

void main() {
  // Membuat objek dari class Person
  Person person = Person("USMAN YAKIN TRIO SINADA", 18);

  print(person.age);        // Menampilkan umur
  person.setName("USMAN");  // Mengubah nama
  print(person.getName());  // Menampilkan nama hasil getter
}

6. NAMED PARAMETER

Named parameter memungkinkan parameter dipanggil berdasarkan nama, membuat kode lebih jelas.

// Nama   : USMAN YAKIN TRIO SINADA
// Nim    : 251410006
// Kelas  : Si2A
class Product {
  String nama; // Nama produk (contoh: Laptop)

  // Constructor
  Product(this.nama);
}

void main() {
  // Membuat objek produk1 dari class Product
  Product produk1 = Product("Usman Website");

  // Menampilkan atribut nama
  print(produk1.nama);
}
